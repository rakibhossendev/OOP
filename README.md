# OOP
This repository for learning OOP 

## L1 
### OOP Summary
- **OOP**: Organizes code around real-world objects
- **4 Pillars**: Encapsulation, Abstraction, Inheritance, Polymorphism
- **Class**: Blueprint for objects
- **Object**: Instance of a class
- **Benefits**: Reusability, Maintainability, Scalability

## L2 
## OOP Key Points Summary
- **Constructor**: `__init__()` method, initializes object attributes
- **Types**:
  - Non-parameterized: `def __init__(self):`
  - Parameterized: `def __init__(self, name, age):`
- **Instance Methods**: Work on object instances, use `self`
- **Use Cases**: Object initialization, attribute setup, behavior definition
